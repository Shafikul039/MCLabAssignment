# MCLabAssignment
Mobile Computing Lab - Flutter UI

This project is a simple Flutter application created for Mobile Computing Lab assignment. It demonstrates basic UI design using Flutter and Dart.

✨ Features : <br>
-> A top AppBar with date <br>
-> A horizontal colored section (ID, Course, Location) <br>
-> A centered information box showing student ID <br>
-> Student name displayed below the card <br>

🛠️ Built With <br>
Flutter <br>
Dart <br>

Material Design <br>
